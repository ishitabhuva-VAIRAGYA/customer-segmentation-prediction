# Customer Segmentation & Prediction using Advanced ML

## 📌 Project Overview

This project applies **unsupervised and supervised machine learning techniques** to segment customers and build prediction models for each segment.

### Objectives

* Perform customer segmentation using clustering
* Compare K-Means, Hierarchical, and DBSCAN
* Build Random Forest prediction models for each segment
* Apply hyperparameter tuning
* Evaluate models using multiple metrics
* Generate business recommendations

---

## 📂 Dataset

Dataset used: `customer_churn.csv`

* Rows: 500
* Columns: 4

Example columns:

* Age
* Annual Income
* Spending Score
* Churn
